# HTML-CSS-JS-Learn
백엔드부터 풀스택까지 길을 넓히기 위해 프론트엔드의 기초인 HTML/CSS/JavaScript를 다루는 Repository

## 초기 세팅
1. VSCode 설치
2. Extensions
 - Live Server/Ritwick Dey : 작성한 HTML 코드를 가상 서버에 띄움
 - Auto Rename Tag/Jun Han : 자동 태그 Rename
 - HTML to CSS autocompletion/solnurkarim : HTML 문서에서 class 속성으로 작성한 값을 연결된 CSS 파일에서 자동 완성
 - HTML CSS Support/ecmel : CSS 파일에서 작성한 클래스나 아이디 선택자 값을 연결된 HTML 문서의 id 속성과 class 속성값으로 적을 때 자동 완성
 - Code Runner/Jun Han : 자바스크립트 코드의 실행을 VSCode 내부에서 할 수 있게 지원
 - Korean Language Pack for Visual Studio Code/Microsoft : 메뉴와 설명을 한글로 변경
